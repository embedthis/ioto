# Claude Sub-Agents

This directory contains definitions for specialized Claude sub-agents used in the Ioto Device Agent project.

## Purpose

Sub-agents are specialized AI assistants configured for specific tasks such as:

- Code review
- Security analysis
- Documentation generation
- Test writing
- Performance optimization

## Format

Each agent definition should be a `.md` file with:

- Agent name and purpose
- Specialized instructions
- Tool access requirements
- Example use cases
