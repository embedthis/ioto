# Reusable Prompts

This directory contains reusable prompts for common development tasks with Claude Code.

## Purpose

Prompts provide consistent instructions for:

- Code review guidelines
- Security analysis templates
- Documentation standards
- Test generation patterns

## Format

Each prompt should be a `.md` file containing:

- Prompt name and purpose
- The prompt text
- Example usage
- Expected outputs

## Usage

Prompts can be referenced in conversations or incorporated into commands and workflows.
