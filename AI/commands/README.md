# Custom Slash Commands

This directory contains custom slash command definitions for Claude Code.

## Purpose

Slash commands provide shortcuts for common development tasks:

- `/build` - Build the project
- `/test` - Run tests
- `/release` - Create a release
- `/doc` - Generate documentation

## Format

Each command should be a `.md` file containing the command prompt that will be expanded when the command is invoked.

## Usage

Commands can be invoked in Claude Code using `/command-name`.
