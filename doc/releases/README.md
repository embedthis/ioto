# Release Notes

This directory contains release notes for each version of the Ioto Device Agent.

## Format

Each release should have a file named `X.Y.Z.md` containing:

- Version number and release date
- Summary of changes
- New features
- Bug fixes
- Breaking changes (if any)
- Migration notes (if applicable)
